# Air Elemental Parent Evaluation

## Selected Production Set

- Bestiary light: `air-elemental-bestiary-light-painterly-vortex.png`
- Silhouette light: `air-elemental-silhouette-light-readable-vortex.png`
- Bestiary dark: `air-elemental-bestiary-dark-storm-vortex.png`
- CRPG dark: `air-elemental-crpg-dark-swooping-vortex.png`

## Evaluation

The selected four-lane set matches the Pathfinder air elemental reference well. All four images present a living air and whirlwind creature with a smoky bird-like upper suggestion, dark hollow face, gray-white wind ribbons, and a tornado-column body. None of the selected production images have solid legs, clothing, armor, weapons, or fire/water/earth elemental drift.

Thumbnail readability is strongest in the silhouette light and CRPG dark lanes. The bestiary light lane is closest to the Pathfinder page's pale parchment presentation. The bestiary dark lane gives the most balanced dark-theme site asset, keeping the gray-white ribbons visible without turning the creature into fire or misty water.

## Rejected Outputs

The `air-elemental-rejected-default-*` files are preserved only as copied alternates from the user-specified default generation folder. They should not be used for production selection because they drift into either a humanoid robed caster with staff or an aquatic serpentine creature. Both directions violate the Air elemental constraints.

## Final Recommendation

Use the four selected production files listed above. If only one light and one dark image are needed downstream, prefer:

- Light: `air-elemental-silhouette-light-readable-vortex.png`
- Dark: `air-elemental-bestiary-dark-storm-vortex.png`
